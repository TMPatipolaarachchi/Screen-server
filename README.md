## Description
A fullscreen screen saver application that displays:
- **Analog Clock** with hour, minute, and second hands
- **Date** in full format (e.g., Monday, February 10, 2026)
- **Digital Time** below the clock

## Language Used
**Python** with **tkinter** (built-in GUI library)

## Features
- ‚úÖ Fullscreen mode
- ‚úÖ Analog clock with moving hands
- ‚úÖ White clock face on black background
- ‚úÖ Red second hand for easy visibility
- ‚úÖ Date display below the clock
- ‚úÖ Digital time display
- ‚úÖ Press **ESC** or **click anywhere** to exit

## Requirements
- Python 3.x (tkinter is included by default)

## Installation Steps

### Step 1: Install Python (Required)
1. Open your web browser and go to: **https://www.python.org/downloads/**
2. Click the **"Download Python 3.x"** button (latest version)
3. Run the downloaded installer
4. ‚úÖ **CRITICAL**: Check the box **"Add Python to PATH"** at the bottom of the installer
5. Click **"Install Now"**
6. Wait for installation to complete
7. Click **"Close"**

### Step 2: Verify Python Installation
1. Open Command Prompt (search "cmd" in Start menu)
2. Type: `python --version`
3. You should see: `Python 3.x.x`
4. If you get an error, restart your computer and try again

### Step 3: Run the Screen Saver

**Easy Method (Recommended):**
- Double-click the file **`run_screensaver.bat`** in the SS folder

**Manual Method:**
1. Open Command Prompt
2. Navigate to this folder:
   ```
   cd C:\Users\thari\Desktop\SS
   ```
3. Run:
   ```
   python screensaver.py
   ```

### Optional: Create Desktop Shortcut
1. Right-click on **`run_screensaver.bat`**
2. Select **"Create shortcut"**
3. Drag the shortcut to your Desktop
4. Rename it to "Screen Saver"
5. Double-click anytime to run!

## Controls
- **ESC key**: Exit the screen saver
- **Mouse click**: Exit the screen saver
